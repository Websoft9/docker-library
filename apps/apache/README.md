# Apache on Docker  

This is an **[Docker Compose template](https://github.com/Websoft9/docker-library)** powered by [Websoft9](https://www.websoft9.com) based on Docker for Apache:


 - community:  2.4, 2.2, latest


## System Requirements

The following are the minimal [recommended requirements](https://github.com/apache/docker#recommended-system-requirements):

* **RAM**: 1 GB or more
* **CPU**: 1 cores or higher
* **Disk**: at least 2 GB of free space
* **bandwidth**: more fluent experience over 100M
