# HAProxy on Docker  

This is an **[Docker Compose template](https://github.com/Websoft9/docker-library)** powered by [Websoft9](https://www.websoft9.com) based on Docker for HAProxy:


 - community:  3.2, 3.1, 3.0, 2.9, 2.8, 2.7, 2.6, 2.5, 2.4, 2.3, 2.0


## System Requirements

The following are the minimal [recommended requirements](https://hub.docker.com/_/haproxy):

* **RAM**: 1 GB or more
* **CPU**: 1 cores or higher
* **Disk**: at least 2 GB of free space
* **bandwidth**: more fluent experience over 100M
