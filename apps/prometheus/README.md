# Prometheus on Docker  

This is an **[Docker Compose template](https://github.com/Websoft9/docker-library)** powered by [Websoft9](https://www.websoft9.com) based on Docker for Prometheus:


 - community:  v3.2.1, latest


## System Requirements

The following are the minimal [recommended requirements](https://hub.docker.com/r/prom/prometheus/):

* **RAM**: 1 GB or more
* **CPU**: 1 cores or higher
* **Disk**: at least 1 GB of free space
* **bandwidth**: more fluent experience over 100M
