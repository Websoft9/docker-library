# Jenkins on Docker  

This is an **Cloud Native solution** powered by [Websoft9](https://www.websoft9.com) based on Docker for Jenkins:

 - community:  2.426, latest


## System Requirements

The following are the minimal [recommended requirements](https://www.jenkins.io/doc/book/installing/docker/):

* **RAM**: 2 GB or more
* **CPU**: 1 cores or higher
* **Disk**: at least 1 GB of free space
* **bandwidth**: more fluent experience over 100M
