# Ansible on Docker  

This is an **[Docker Compose template](https://github.com/Websoft9/docker-library)** powered by [Websoft9](https://www.websoft9.com) based on Docker for Ansible:


 - community:  2.16.1


## System Requirements

The following are the minimal [recommended requirements](https://docs.ansible.com/ansible/latest/installation_guide):

* **RAM**: 1 GB or more
* **CPU**: 1 cores or higher
* **Disk**: at least 1 GB of free space
* **bandwidth**: more fluent experience over 100M
